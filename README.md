# 🏔️ SewaAja - Platform Penyewaan Alat Pendakian

---

## 📌 Deskripsi

**SewaAja** adalah platform digital berbasis web yang memudahkan para pecinta alam dan pendaki untuk **menyewa perlengkapan pendakian** secara praktis tanpa harus membeli peralatan mahal. Aplikasi ini menghubungkan antara pemilik alat (admin) dan pendaki (user) dalam satu ekosistem yang aman, modern, dan terpercaya.

Dengan konsep *sharing economy*, SewaAja mendukung gaya hidup berkelanjutan dengan mengurangi pembelian peralatan yang hanya dipakai sesekali.

---

## 🚀 Fitur Utama

| No | Fitur | Keterangan |
|----|-------|------------|
| 1 | **Katalog Alat Pendakian** | Menampilkan daftar perlengkapan gunung dengan gambar |
| 2 | **Filter Kategori** | Menyaring produk berdasarkan kategori |
| 3 | **Pencarian Produk** | Cari perlengkapan berdasarkan nama |
| 4 | **Login Multi-Role** | Admin dan User dengan akun berbeda |
| 5 | **CRUD Produk** | Admin bisa Tambah, Edit, Hapus alat pendakian |
| 6 | **Form Sewa** | Isi nama, WhatsApp, alamat, dan tanggal sewa |
| 7 | **Pilihan Metode Pembayaran** | QRIS atau Bayar di Tempat |
| 8 | **QR Code Pembayaran** | Generate QR Code untuk pembayaran |
| 9 | **Konfirmasi WhatsApp** | Kirim bukti pembayaran ke WhatsApp admin |
| 10 | **Diskon Otomatis** | Diskon 10% (3+ hari) dan 15% (7+ hari) |
| 11 | **Manajemen Penyewaan Admin** | Setujui, tolak, selesaikan sewa |
| 12 | **Status Otomatis** | Status barang berubah otomatis |
| 13 | **Dashboard Statistik Admin** | Total, tersedia, disewa, pendapatan |
| 14 | **Dark Mode** | Tampilan gelap untuk kenyamanan malam |
| 15 | **Glassmorphism Design** | Tampilan modern dengan efek kaca |
| 16 | **Responsive HP** | Bisa diakses dari HP, tablet, laptop |
